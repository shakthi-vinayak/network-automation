This repository is a backend/network automation platform built with Ansible, Python, Terraform, and Jinja2 templates. It contains no frontend code (no CSS, SCSS, HTML, JSX/TSX, Vue, Svelte, or any browser-facing assets). The only UI-related references are to Grafana/Prometheus dashboards defined as code (YAML/Terraform), which are configuration files rather than styled UI components. There is no CSS methodology, design token system, component library, theme, or responsive strategy in this repo.
